# Workspace Automator 🚀

Una herramienta GUI en Python para automatizar la apertura de entornos de desarrollo (VS Code, Apps y Webs) con un solo clic. Ideal para desarrolladores que quieren empezar a trabajar sin perder tiempo abriendo programas uno a uno.

## ✨ Características

* **Categorías de Trabajo:** Crea perfiles distintos (ej: "Backend", "Frontend", "Ocio").
* **Soporte Multi-tipo:**
* 🖥️ **Apps (.exe):** Lanza ejecutables de escritorio.
* 🌐 **Web:** Abre múltiples URLs en pestañas de tu navegador predeterminado.
* 👨‍💻 **VS Code:** Abre carpetas de proyecto directamente en el editor.


* **Persistencia:** Guarda tus configuraciones automáticamente en un archivo local (`.json`).
* **Interfaz Moderna:** Construido con `CustomTkinter` (soporte nativo de Modo Oscuro).

## 📦 Instalación

1. Clona el repositorio:
git clone [https://github.com/TU-USUARIO/workspace-automator.git](https://www.google.com/search?q=https://github.com/TU-USUARIO/workspace-automator.git)
cd workspace-automator
2. Instala las dependencias:
pip install -r requirements.txt

## 🚀 Uso

Ejecuta el script principal desde la terminal:

python launcher_pro.py

1. Crea una **Nueva Categoría** (ej: *Trabajo*).
2. Añade tus herramientas (Apps, Webs o Proyectos).
3. Presiona **🚀 LANZAR ENTORNO**.

## 🛠️ Tecnologías

* Python 3.x
* [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)
* JSON (Almacenamiento local)
