Actúa como un Arquitecto de Software y Desarrollador Full-Stack experto en C# (.NET/WPF), React y APIs nativas. Basándote en la estructura de mi proyecto 'Workspace Launcher' (Backend C# + Frontend React/Vite con WebView2), quiero desarrollar 'CustomZoneEngine', un motor de plantillas propio que replique la funcionalidad y UX de Microsoft FancyZones, pero dándote libertad creativa para adaptar la UI/UX al estilo de mi aplicación actual. Requisitos del Frontend (React): 1) Un gestor de Layouts que permita seleccionar el monitor en la parte superior y elegir entre plantillas predefinidas (Filas, Columnas, Cuadrícula) o crear Diseños Personalizados. 2) Un Editor Visual a pantalla completa (Overlay Canvas) con las siguientes mecánicas interactivas: usar 'Shift + Clic' para dividir una zona existente (vertical u horizontalmente), hacer clic y arrastrar para combinar zonas, arrastrar los divisores para redimensionarlas, y un panel de configuración con sliders para ajustar el espaciado (márgenes) entre las zonas. Requisitos del Backend y UX (C#): 1) Intercepción global: al arrastrar cualquier ventana de Windows y mantener 'Shift', debe mostrarse el overlay de las zonas y, al soltar, la ventana debe auto-ajustarse a la zona seleccionada. 2) Compatibilidad total con mi sistema actual de apilamiento y rotación, integrándose con ZoneCycler.cs y ZoneStack.cs. Requisitos Arquitectónicos: 1) Retrocompatibilidad: Añadir un toggle en ConfigPanel.jsx y ConfigManager.cs para que el usuario elija entre usar el módulo antiguo [FancyZones] de PowerToys o este nuevo 'CustomZoneEngine'. 2) Future-proofing (Multiplataforma): Aplica principios SOLID y el patrón Adapter. Aísla todas las llamadas a Win32 (carpeta [NativeInterop]) detrás de interfaces (ej. IWindowArranger, IZoneDetector). Esto es obligatorio para que en un futuro la app pueda compilarse para macOS o Linux implementando dichas interfaces. Proporcióname la arquitectura, el flujo del WebBridge.cs, la estructura de componentes React para el editor interactivo y la lógica en C#.
